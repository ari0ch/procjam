This game was built for #procjam 2014. Procjam is a game jam about procedural generation.

The levels in this game are generated procedurally. When each level loads you will see the level being generated by drones created to visualise the procedural level generator.

The drones follow 3 steps - 1. create mines, 2. connect them with corridoors, 3. add water and other liquids, 4. add objects, 5. add enemies, 6. add the player, 7. add the exit.
